# Film Reviews

## Project Description

Film Reviews is a web application where users can browse movies, read reviews, and submit their own reviews.

## Development Environment

- Python 3.9
- uv
- Git
- VS Code

## Tools

- Black (formatter)
- Ruff (linter)
- Pytest (testing)
- Pytest-Cov (coverage)

## Setup

```bash
uv venv
source .venv/bin/activate
uv pip install black ruff pytest pytest-cov
